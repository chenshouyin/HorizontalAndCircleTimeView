####水平刻度尺基本功能
* 水平刻度划分为60份，代表60分钟，可自己修改
* 刻度尺上课自定义数字标识
* 可区分大小刻度
* 可以拖动游标前进后退
* 刻度值变化回调

####圆形钟表刻度尺基本功能
* 圆弧划分为24等份，代表24小时,可自己修改
* 形钟表刻度尺外围可自定义数字标识
* 可区分大小刻度
* 可点击可拖动
* 可以前进后退
* 刻度值变化回调
